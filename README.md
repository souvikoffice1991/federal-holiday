# Project Title

Federal-Holiday project

## Description

project to display list of holidays by country
add or update the list of holidays for each country

## Getting Started

### Dependencies

* will need jdk 17 version or higher
  

### Installing

* download as zip
* unzip the folder
* got to the root folder which has build.gradle file in it
* from terminal run ./gradlew clean build
  

### Executing program

* after install from commandline run following commands
  * java -jar build/libs/holidaycalendar-1.0-SNAPSHOT.jar
